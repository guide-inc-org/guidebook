# 折りたたみチャプター

サイドバーナビゲーションはネストされたコンテンツの折りたたみをサポートしています。

## 動作

サブアイテムを持つチャプターは、矢印アイコンをクリックして展開/折りたたみできます。

```markdown
# 目次

* [はじめに](README.md)
* [第1章](chapter1/README.md)
  * [セクション 1.1](chapter1/section1.md)
  * [セクション 1.2](chapter1/section2.md)
* [第2章](chapter2.md)
```

この例では、「第1章」に展開/折りたたみの矢印が表示されます。

## 状態の保存

サイドバーの状態は `localStorage` に保存されるため、展開/折りたたみ状態はページリロード後も維持されます。

## 折りたたみを無効化

```json
{
    "plugins": ["-collapsible-chapters"]
}
```
