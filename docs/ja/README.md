# はじめに

**guidebook** は、Rust で書かれた高速な HonKit/GitBook 互換の静的サイトジェネレーターです。

## なぜ guidebook？

- **高速** - Rust による最高のパフォーマンス
- **シンプル** - 単一バイナリ、依存関係なし
- **互換性** - HonKit/GitBook のドロップイン置き換え
- **自己更新** - `guidebook update` でアップデート

## クイック例

```bash
# インストール
curl -fsSL https://raw.githubusercontent.com/guide-inc-org/guidebook/main/install.sh | sh

# プレビュー
guidebook serve

# ビルド
guidebook build
```
