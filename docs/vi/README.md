# Giới thiệu

**guidebook** là trình tạo trang web tĩnh nhanh, tương thích với HonKit/GitBook, được viết bằng Rust.

## Tại sao chọn guidebook?

- **Nhanh** - Được xây dựng bằng Rust để có hiệu suất tối đa
- **Đơn giản** - Một file nhị phân duy nhất, không phụ thuộc
- **Tương thích** - Thay thế trực tiếp cho HonKit/GitBook
- **Tự cập nhật** - Cập nhật bằng `guidebook update`

## Ví dụ nhanh

```bash
# Cài đặt
curl -fsSL https://raw.githubusercontent.com/guide-inc-org/guidebook/main/install.sh | sh

# Xem trước
guidebook serve

# Build
guidebook build
```
